# 见习侦探的暑期事件簿 -- 使用说明书

> 侦探主题暑假学习打卡应用 | 初一升初二 | iPad 移动端优先 | 2026.07.13 ~ 2026.08.31

---

## 一、应用概览

把暑假学习变成一场侦探冒险。每天完成学习任务获得星星，收集线索侦破案件，解锁成就提升侦探等级。

**四个主标签页：**

| 标签 | 图标 | 功能 |
|------|------|------|
| 今日搜证 | 放大镜 | 每日打卡主页，完成当天任务 |
| 案卷归档 | 书 | 查看历史记录，搜索筛选 |
| 名侦探履历 | 图表 | 统计数据、成就墙、日历热力图 |
| 密文破译 | 对话气泡 | 英语单词背诵卡片 |

**顶部导航栏**：左右箭头切换日期，日历按钮回到今天。不可操作未来的日期。

---

## 二、今日搜证（每日打卡）

每天的打卡主页，按学科分类展示当日任务。

### 2.1 头像上传

页面顶部有侦探头像区，点击可上传自定义画像（建议正方形，iPad 拍照或相册选取）。

### 2.2 每日金句 + 心情

- **抽签**：点击抽签按钮随机获得一句励志金句，每天一次
- **心情**：选择今天的心情（开心 / 平静 / 疲惫 / 加油），会显示在日历热力图上

### 2.3 每日密令

系统每天自动生成一条挑战密令（如"今天多完成1项背诵任务"），完成后获得额外星星。

### 2.4 背诵任务（古籍破译 + 密文翻译）

按学科分块展示，每块默认显示前 3 条未完成任务，点击"展开全部"查看所有。

- **古文背诵（古籍破译）**：八上 27 篇古诗文，点击卡片查看全文
- **英语背诵（密文翻译）**：七下 8 单元 + 八上 8 单元，共 16 组单词
- **每日固定任务**：阅读理解、时文阅读、推荐书目等，周六英语自动切换为作文+听力

**操作**：点击星星按钮打卡，已完成的点勾号撤销。

### 2.5 学科任务（密码解析 + 现场重建）

数学和物理的章节小节，按章节分组折叠展示。旅游期（8/1-8/10）自动隐藏。

**操作**：点击章节标题展开小节列表，点击具体小节打卡。

### 2.6 体能训练

四项运动打卡：游泳、拉伸、跑步、跳绳。

### 2.7 暑假作业

按学科（语文、数学、英语、道法、历史、体育）分组的作业清单，可折叠查看。点击作业项查看详情，点击星星打卡。

### 2.8 特别行动

自定义任务，每天最多添加 5 个。点击"+ 添加特别行动"输入名称和备注即可。

### 2.9 探案笔记（学习日记）

每天可以写学习心得，支持编辑修改。笔记也会展示在周日休息页和履历页中。

### 2.10 推理破案

当收集到足够线索后，页面会出现"推理准备就绪"卡片。点击**开始推理**进入破案弹窗，根据线索选择答案。答对案件告破，答错可以重试。

## 三、密文破译（单词背诵卡）

翻转卡片式的英语单词学习工具。

### 3.1 基本操作

- **翻转卡片**：点击卡片翻面查看释义，再点击翻回
- **切换单词**：点击左右箭头或左右滑动卡片
- **单元选择**：顶部下拉框选择要背诵的单元（七下 U1-U8 + 八上 U1-U8）

### 3.2 掌握度标记

- 点击"我会了"标记单词为已掌握
- 默认开启"未掌握"模式，已掌握的单词自动跳过
- 可切换到"全部"模式查看所有单词
- 每个单元旁边显示掌握进度（如 15/28）

---

## 四、案卷归档（历史记录）

查看所有已完成的任务记录。

- **筛选**：按学科筛选、按关键词搜索
- **排序**：最新优先 / 最早优先 / 星星最多
- **概览**：顶部显示总完成项数、星星数、打卡天数，以及各学科完成进度条
- **撤销**：每条记录可撤销（星星也会收回）
- **证物陈列**：可查看收集到的盲盒证物

---

## 五、名侦探履历（统计仪表盘）

### 5.1 侦探等级

根据累计星星数自动升级，共有多个侦探等级（见习侦探 → 初级侦探 → ... → 传奇侦探）。显示当前等级图标和距离下一级的进度条。

### 5.2 关键数据

累计星星、活跃天数、连续打卡天数、最长连续打卡。

### 5.3 谜案侦破

显示当前侦办中的案件进度（线索收集情况）和已破案件列表。每次打卡有概率获得线索。

### 5.4 侦探之旅（日历热力图）

七月和八月双月并排显示，每天一个小格子：
- **绿色格子**：有打卡的日子，颜色越深星星越多
- **黄色格子**：旅行期
- **灰色格子**：休息日或未打卡
- **紫色边框**：今天

格子内会显示当天选择的心情图标。点击格子可看到详细信息。

### 5.5 每周小结

按周汇总：打卡天数、获得星星、破案数量。最近 6 周，本周有特殊标识。

### 5.6 学科进度

环形进度图展示各学科的完成率、已打卡项数和星星数。

### 5.7 成就徽章

完成特定目标解锁徽章（如连续打卡 7 天、累计 100 星等），解锁时弹窗展示。

### 5.8 探案笔记

按时间倒序展示最近 14 天的学习日记。

### 5.9 证据陈列室

展示收集到的盲盒证物（最多 12 件）。

### 5.10 数据管理

- **导出备份**：下载包含所有打卡记录、成就、日记的 JSON 文件
- **导入恢复**：从备份文件恢复数据（会覆盖当前数据，操作前会确认）

## 六、特殊日期规则

| 日期类型 | 规则 |
|----------|------|
| **周日** | 全休日，不显示任何任务，展示休息页面和本周统计 |
| **旅游期（8/1-8/10）** | 数学和物理章节隐藏，背诵+每日固定+体育继续 |
| **周六** | 英语每日任务"阅读理解"自动替换为"作文+听力" |
| **未来日期** | 不可打卡，按钮置灰 |

## 七、星星与奖励系统

### 7.1 星星获取

每完成一项任务获得 1 颗星星，每天最多 20 颗。完成每日密令有额外奖励。

### 7.2 盲盒（证物收集）

完成任务达到一定条件自动触发盲盒：
- **日常证物**：累积完成一定数量任务时触发
- **关键证据**：特殊里程碑触发
- **额外发现**：随机惊喜

### 7.3 复习机制

背诵类任务（古文、英语单词）采用间隔复习，完成首次打卡后在第 1、2、4、7、15 天自动安排复习，确保记忆巩固。

---

## 八、数据存储说明

- 所有数据存储在浏览器 IndexedDB 中，**不会上传到任何服务器**
- 清空浏览器缓存/数据会丢失所有记录
- **建议定期导出备份**（在"名侦探履历"页底部）
- 换设备可使用导入功能迁移数据
- GitHub Pages 部署的版本与本地版数据独立，各自存储在对应浏览器的 IndexedDB 中

## 九、快捷操作

| 操作 | 方式 |
|------|------|
| 切换日期 | 顶部左右箭头 / 键盘左右方向键 |
| 回到今天 | 顶部日历按钮 |
| 翻转单词卡 | 点击卡片 |
| 切换单词 | 点击箭头 / 左右滑动卡片 |
| 查看任务全文 | 点击背诵任务卡片 |
| 硬刷新（清除缓存） | Cmd + Shift + R（Mac）/ Ctrl + Shift + R（Win） |

## 十、家长使用建议

1. **初次设置**：打开应用后，和孩子一起浏览各个标签页，了解侦探主题的故事设定
2. **每日节奏**：早上查看今日密令 → 白天完成各项任务打卡 → 晚上写探案笔记
3. **定期检查**：每周查看"名侦探履历"的每周小结和日历热力图，了解孩子的学习节奏
4. **激励机制**：利用侦探等级和成就徽章激发孩子的打卡动力
5. **数据备份**：每周导出一次数据备份，防止意外丢失
