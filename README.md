# 见习侦探的暑期事件簿

一个侦探主题的暑假学习打卡应用，专为初一升初二学生设计。把每天的学习任务变成"探案"，让孩子在破案的乐趣中完成暑假作业。

---

## 为什么做这个？

暑假 50 天，作业多、任务杂、孩子容易懈怠。我们把它做成了一场侦探游戏——每天打卡就是"收集线索"，集齐线索就能"侦破案件"。从被动完成任务变成主动追案子，学习动机完全不一样了。

---

## 核心亮点

### 侦探破案系统（50 个案件）
不是简单地打完卡就完了。每完成一个学习任务，有 40% 概率掉落一条线索。收集足够的线索后，孩子需要**自己推理**——阅读案情描述和已收集的线索，从三个选项中选出正确答案，才能宣告"CASE CLOSED"。

50 个校园谜案，从"失踪的暑假作业"到"毕业季的最后一案"，难度逐步升级，逻辑清晰、结局温暖，每个案件都有完整的推理闭环。

### 抽签金句 + 每日心情
不是系统自动推送一句鸡汤。孩子每天**自己抽签**——点击红色抽签按钮，从 75 条金句中随机抽取一条，75 条用完才重置。旁边可以选今日心情，😊开心、🤩兴奋、😎自信……9 种情绪，每天记录。心情表情直接叠加在日历上，一眼看到整个暑假的情绪起伏。

### 翻转单词卡片
16 组沪教版英语单词（七下复习 + 八上预习），做成 3D 翻转卡片。正面英文，点击翻面看中文释义，左滑右滑切换单词。学会了点"我会了"，自动跳过，专注背不会的。所有单词来自真实教材，不是随便拼凑的。

### 暑假作业清单（21 项）
语文 7 项、数学 7 项、英语 3 项、道法 1 项、历史 1 项、体育 2 项。不是简单的标题列表——点击每项可以查看从 PDF 提取的**完整要求**（尺寸、格式、提交方式等），不用翻原始材料。按学科折叠，完成一项勾一项，进度一目了然。

### 学科技能环 + 双月日历
6 个学科做成 SVG 圆形进度环，带光晕效果。七月和八月两个迷你日历并排显示，格子颜色深浅表示当天获得星星数，格子上还叠了心情表情。配合每周小结卡片，孩子的学习节奏一目了然。

### 37 枚侦探主题成就徽章
从"初次登场"到"警界之光"，从"三日追踪"到"侦探之王"（连续打卡 30 天），从"证物碎片"到"传奇侦探"（500 颗星）。每枚徽章都有侦探主题的命名和 icon，给孩子持续的成就感。

## 完整功能清单

### 页面一：今日搜证（主打卡页）

| 功能 | 说明 |
|------|------|
| 自定义头像 | 上传孩子的画像或个人照片，圆形展示 |
| 每日抽签 | 75 条金句随机抽取，不重复，用完自动重置 |
| 每日心情 | 9 种情绪可选，实时记录到日历 |
| 古诗背诵 | 27 篇八上古诗文（文言文 + 诗歌 + 宋词），含全文 |
| 英语单词 | 16 组沪教版词汇（七下 8 组 + 八上 8 组），每组 30-45 词 |
| 每日固定任务 | 语文阅读理解、推荐阅读、英语时文阅读等 |
| 数学/物理章节 | 按课本章节拆分，可折叠展开，逐节打卡 |
| 体育运动 | 游泳、跳绳、篮球等常规项目 |
| 推理按钮 | 线索集齐后出现金色闪烁按钮，点击进入推理弹窗 |
| 暑假作业清单 | 21 项按学科折叠，点击查看完整要求，完成勾选 |
| 特别行动 | 自定义添加当日额外任务 |
| 探案笔记 | 每天写学习日记，支持修改 |
| 盲盒证物 | 完成 3 类任务或获得 5 星触发，收集证物卡片 |

### 页面二：案卷归档（历史记录）

| 功能 | 说明 |
|------|------|
| 全部打卡记录 | 按日期分组，支持撤回 |
| 学科筛选 | 按语文/数学/英语等学科过滤 |
| 关键词搜索 | 搜索任务名称 |
| 排序切换 | 最新优先 / 最早优先 / 星星最多 |
| 证物陈列 | 盲盒收集的证物以 3 列卡片网格展示 |

### 页面三：名侦探履历（统计仪表盘）

| 功能 | 说明 |
|------|------|
| 侦探等级 | 6 级晋升体系（见习→初级→中级→高级→名侦探→传奇） |
| 关键数据 | 累计星星、活跃天数、连续打卡、最长连续 |
| 谜案侦破 | 当前侦办案件进度 + 已破案件列表 |
| 双月日历热力图 | 7 月 + 8 月并排，颜色深浅表示打卡热度，同步显示心情表情 |
| 每周小结 | 最近 6 周，每周的打卡天数、获得星星、侦破案件数 |
| 学科技能环 | 6 个圆形 SVG 进度环，带光晕动画 |
| 37 枚成就徽章 | 3 列网格，含解锁日期 |
| 探案笔记列表 | 最近 14 篇日记展示 |
| 数据备份/恢复 | JSON 导出/导入，支持跨设备迁移 |

### 页面四：密文破译（单词背诵）

| 功能 | 说明 |
|------|------|
| 3D 翻转卡片 | 点击翻转查看释义，0.5 秒动画 |
| 单元选择 | 下拉框 16 组可选，显示已掌握/总数 |
| 已掌握标记 | "我会了"按钮，自动跳过，可切换查看全部 |
| 滑动手势 | 左右滑动切换单词 |
| 上下导航 | 按钮切换 + 进度显示 "5/12" |

| 旅途模式 | 8/1-8/10 旅游期，数学物理章节自动隐藏 |
| 周六特殊 | 英语每日任务自动替换为作文+听力 |

## 适合谁用？

- 初一升初二的学生（内容匹配沪教版八上）
- 希望孩子暑假有规律、不突击的家长
- 喜欢游戏化学习、需要外部激励的孩子
- 老师布置了分散的暑假作业、需要统一管理的家庭
